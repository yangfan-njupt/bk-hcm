### 描述

- 该接口提供版本：v9.9.9+。
- 该接口所需权限：业务访问。
- 该接口功能描述：查询业务下没有应用此策略的账号ID列表。根据策略库的使用业务列表匹配二级账号的管理业务（仅当前业务），筛选出二级账号后，过滤掉已在权限模板表中关联此策略库的二级账号，全量返回未应用此策略的二级账号ID列表，不分页。

### URL

GET /api/v1/cloud/bizs/{bk_biz_id}/vendors/{vendor}/permission_policy_libraries/{id}/unapplied_account_ids

### 路径参数

| 参数名称      | 参数类型   | 必选 | 描述                        |
|-----------|--------|----|---------------------------|
| bk_biz_id | int64  | 是  | 业务ID                      |
| vendor    | string | 是  | 云厂商（枚举值：tcloud） |
| id        | string | 是  | 策略库ID                     |

### 调用示例

```json
```

### 响应示例

```json
{
  "code": 0,
  "message": "",
  "data": {
    "account_ids": [
      "00000001",
      "00000002",
      "00000003"
    ]
  }
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
| data    | object | 响应数据 |

#### data

| 参数名称        | 参数类型         | 描述                                                              |
|-------------|--------------|------------------------------------------------------------------|
| account_ids | string array | 未应用此策略的二级账号ID列表（已去重，仅包含管理业务为当前业务的账号，全量返回，不分页） |
